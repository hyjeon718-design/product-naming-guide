# Ingredient Brand 가이드

AIWORKX 기술 브랜드("Powered by X™") 검토 기준입니다.
제품명(Category — ProductName)과 별개 레이어로 관리됩니다.

---

## Ingredient Brand란?

Intel Inside처럼 제품 내부에 탑재된 핵심 기술을 외부에 브랜딩하는 방식.
AIWORKX는 **ToD™(Task-oriented Deterministic AI)** 엔진을 대표 Ingredient Brand로 운영합니다.

표기 형식:
```
AIWORKX [Category] [ProductName] — Powered by [IngredientBrand]™
예시: AIWORKX Agent Ensemble — Powered by Linea™
```

---

## 적용 시점

**신제품/솔루션 개발·출시 시** 제품명과 함께 명시합니다.
- 개발 단계: 내부 문서·기술 스펙에 표기
- 출시 단계: 마케팅 자료, 제품 소개 페이지, 보도자료에 반드시 포함
- 사후 추가: 기존 제품에 해당 기술 탑재 시 버전 업데이트와 함께 명시

---

## 기타 카테고리 진입 조건

0단계에서 카테고리 = "기타"를 선택하는 경우:
- 제품명이 아닌 **기술 브랜드 자체**가 검토 대상
- "Powered by X™" 형태로 여러 제품에 걸쳐 사용될 기술
- 예: ToD™ 엔진, 추후 신규 AI 기반 기술 레이어

---

## 평가 기준 (4가지)

| 기준 | 설명 | 확인 방법 |
|------|------|----------|
| 1. 조어 여부 | 신조어/조합어일수록 상표 등록 유리 | 사전 검색 + KIPRIS |
| 2. 발화 자연스러움 | "Powered by X™" 문장에서 자연스럽게 읽히는가 | 소리 내어 읽기 테스트 |
| 3. Generic term 회피 | 학술·업계 공용어 아닌가 | Google Scholar + 업계 용어집 |
| 4. 포괄성 | 기술 확장 시에도 적합한 이름인가 | 향후 로드맵과 대조 |

---

## Generic Term 판단 기준

다음 중 하나라도 해당하면 Generic Term 경고 🔴:
- Google Scholar에서 100편 이상 논문에 사용
- Wikipedia 독립 항목으로 등재
- 주요 오픈소스 프레임워크 이름으로 사용 중
- 업계 표준 용어집(IEEE, ACM 등)에 등재

---

## ToD™ 현황

| 항목 | 상태 |
|------|------|
| 공식명 | ToD™ (Task-oriented Deterministic AI) |
| Generic term 이슈 | 🔴 NLP 학술계 "Task-Oriented Dialogue" 약어와 충돌 |
| 현재 사용 | 내부 개발 단계 한정 사용 허가 |
| 권고사항 | 외부 출시 전 대안 검토 (Linea™ 등) |
| NRB 상정 여부 | 미완 — 승인 전 외부 표기 금지 |
